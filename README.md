# TelegramBot-Delay
Un BOT Telegram permetant de planifier l'envoi de message

## Telegram - Contact
You just have to add :
> **MrDelayBot**

## Commands
- /help 
- /start

Print help message
- Hello !

Say hello to the bot !

## Messages management

Example :
> 5mn Lorem ipsum

The message "Lorem ipsum" will be sent back in 5 minutes
